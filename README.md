# 🎵 Prueba‑M2

Repositorio con un sitio web estático que presenta una colección de discos musicales. Incluye páginas para distintos discos, sección “Quién soy”, contacto, estilos, y una carpeta de imágenes.

---

## 🗂️ Estructura de archivos y carpetas

```
Prueba‑M2/
├─ imágenes/               # Portadas y gráficos utilizados
├─ index.html              # Página principal con lista de discos
├─ about.html              # “Quién soy” o descripción del autor
├─ contacto.html           # Formulario o información de contacto
├─ disco1.html … disco9.html  # Páginas individuales para cada disco
├─ style.css               # Estilos CSS principales
└─ style1.css              # Estilos alternativos
```

---

## 🚀 Uso

1. Clona el repositorio:
   ```bash
   git clone https://github.com/juan35marin/Prueba-M2.git
   cd Prueba-M2
   ```
2. Abre `index.html` en tu navegador. Puedes navegar entre los discos, ver la descripción, sobre el autor o contactarte.

---

## 🎨 Personalización

- Añade o modifica discos creando nuevos archivos `discoX.html` y agregando sus imágenes en la carpeta `imágenes/`.
- Edita `style.css` y `style1.css` para cambiar diseño, tipografía, colores, o hacerlo responsivo.
- Ajusta `contacto.html` si deseas integrar un formulario funcional.

---

## 📌 Recomendaciones futuras

- Agregar archivos JavaScript para interacción (navegación dinámica, sliders, formularios).
- Convertir en sitio responsivo usando media queries en CSS.
- Implementar un formulario real con back-end (PHP, Node.js, etc.).
- Optimizar las imágenes para mejorar tiempos de carga.
- Automatizar la generación de páginas de discos con un generador estático (Jekyll, Hugo…).
